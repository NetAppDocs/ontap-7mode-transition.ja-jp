= ネームサービスの移行準備をしています
:icons: font
:imagesdir: ../media/


[role="lead"]
DNS 、 LDAP 、 NIS 、ホスト、ネームサービススイッチなどのネームサービス設定 UNIX ユーザおよびグループおよびネットグループの構成は 7-Mode Transition Tool で移行されます。ネームサービス構成を移行する前に、いくつかの考慮事項を理解しておく必要があります。
