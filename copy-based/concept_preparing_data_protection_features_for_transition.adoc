= データ保護機能の移行準備をしています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
7-Mode の SnapMirror 関係を移行するには、いくつかの手順を手動で実行する必要があります。また、データ保護関係のうち、移行対象としてサポートされるものとサポートされないものを把握しておく必要があります。
