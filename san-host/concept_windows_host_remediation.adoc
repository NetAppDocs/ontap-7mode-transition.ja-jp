= Windows ホストの修正
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
7-Mode Transition Tool （ 7MTT ）を使用して Data ONTAP 7-Mode から clustered Data ONTAP に移行する場合は、特定の手順を実行して Windows ホストで移行準備を行う必要があります。また、カットオーバーフェーズに備えて特定の手順を実行し、移行後に Windows ホストをオンラインにする必要があります。
